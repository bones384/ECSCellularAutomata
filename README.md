# Projekt PK2
# Entity Component System zastosowany w automatach komórkowych
Projekt kompilowany w standardzie C++23, zalecam korzystanie z załączonego pliku CMake, który sam pobierze i załączy do kompilacji najnowszą wersję Rayliba. Wszystkie inne biblioteki są jednonagłówkowe i dołączone do projektu na githubie. Kompilowany na Linuxie używając g++ oraz na Windowsie przy użyciu mingw.
